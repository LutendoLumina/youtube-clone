# 🎬 YouTube Homepage Clone (Zaio Project)

A fully responsive **YouTube Homepage Clone** built using **HTML and CSS
only**.\
This project recreates the core UI and interaction patterns of YouTube,
including a collapsible sidebar, dark mode toggle, hover video previews,
responsive layout, and a Shorts section.

> ⚡ Built with pure HTML + CSS (No JavaScript required for core
> features)

------------------------------------------------------------------------

## 🚀 Features

### ✅ Fully Functional Layout

-   Navigation bar (logo, search bar, mic button, profile section)
-   Collapsible sidebar
-   Video grid layout
-   Shorts section (vertical cards)
-   Scrollable filter chip bar
-   Styled sidebar footer

### 🌙 Dark Mode (CSS-Only)

-   Toggle switch in the navbar
-   Uses CSS variables (`:root`)
-   No JavaScript used
-   Smooth theme transitions

### 🎥 Hover Video Preview

-   Standard videos show `.webp` preview on hover
-   Shorts autoplay muted `.mp4` preview
-   Smooth fade-in effect
-   Fully CSS-driven

### 📱 Fully Responsive Design

  Device          Layout
  --------------- ---------------------------------
  Large Screens   3 videos per row
  Tablets         2 videos per row
  Phones          1 video per row
  Shorts          5 → 4 → 3 → 2 per row
  Mobile          Sidebar becomes slide-in drawer

## 🎯 Key Concepts Implemented

-   Sidebar collapse using checkbox hack
-   CSS Grid video layout
-   Responsive breakpoints
-   Hover previews with layered positioning
-   Styled scrollbars and smooth transitions

## 🧪 How to Run

1.  Download or clone the repository
2.  Open `index.html` in your browser
3.  Toggle:
    -   🌙 Dark Mode
    -   📂 Sidebar collapse
    -   🎥 Hover previews

No installation required.
